# SkiNix Skincare Shop

**SkiNix** is a fully responsive e-commerce website.  
Responsive for all devices, built using **HTML**, **CSS**, and **JavaScript**.

## Features

- Video background on hero banner
- Add to Cart system with dynamic total and quantity display
- Add to Favorites system with visual feedback
- Mobile-friendly and scrollable product display
- Smooth animations and hover effects
- Checkout modal with order summary

## Technologies Used

- HTML5
- CSS3 (with custom properties and responsive design)
- JavaScript (DOM manipulation, event listeners, and interactivity)
